# [P1305 新二叉树](https://www.luogu.com.cn/problem/P1305)

## 题目描述

输入一串二叉树，输出其前序遍历。

## 输入格式

第一行为二叉树的节点数 n 。(1 ≤ n ≤ 26)

后面 n 行，每一个字母为节点，后两个字母分别为其左右儿子。

空节点用 `*` 表示

## 输出格式

二叉树的前序遍历。

## 输入输出样例

**输入 #1**复制

```
6
abc
bdi
cj*
d**
i**
j**
```

**输出 #1**复制

```
abdicj
```
